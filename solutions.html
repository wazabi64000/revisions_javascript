Exercices Solutions

1. Console et variables

let a = 5;
let b = 10;
console.log("Résultat :", a + b);

On déclare deux variables avec let et on utilise console.log() pour afficher la somme.

2. Tableaux et boucles

const nombres = [1,2,3,4,5];
for(let i=0; i<nombres.length; i++){
    console.log(nombres[i]);
}

Boucle for pour parcourir le tableau et afficher chaque élément.

3. Fonctions

function carre(n){
    return n * n;
}
console.log(carre(5)); // 25

La fonction prend un argument et renvoie son carré. On utilise return pour renvoyer la valeur.
