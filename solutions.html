Solutions détaillées - Révision JavaScript

Exercices Solutions

1. Console et variables

let a = 5;
let b = 10;
console.log("Résultat :", a + b);

Explication : On déclare deux variables avec let et on utilise console.log pour afficher leur somme. C’est le premier pas pour comprendre la manipulation des variables et l’affichage en console.

2. Tableaux et boucles

const nombres = [1, 2, 3, 4, 5];
for (let i = 0; i < nombres.length; i++) {
  console.log(nombres[i]);
}

Explication : Un tableau est une collection ordonnée. La boucle for permet d’itérer sur chaque élément. La propriété length retourne le nombre d’éléments.

3. Fonctions

function carre(n) {
  return n * n;
}
console.log(carre(5)); // 25

Explication : Les fonctions prennent des paramètres et retournent des valeurs. Ici, on calcule le carré d’un nombre.

4. Manipulation du DOM

document.querySelector("#monParagraphe").textContent = "Nouveau texte";

Explication : querySelector cible un élément DOM par son ID, et textContent modifie le texte visible. Cela introduit la manipulation du DOM.

5. Événements

document.querySelector("#monBouton").addEventListener("click", () => {
  alert("Bouton cliqué !");
});

Explication : Les événements permettent de rendre la page interactive. Ici, un clic déclenche une alerte.

6. Objets simples

const personne = { nom: "Alice", age: 25 };
console.log(personne.nom);

Explication : Les objets regroupent plusieurs valeurs sous des clés. On peut accéder à chaque propriété via objet.propriété.

7. Méthodes d’objets

const personne = {
  nom: "Alice",
  saluer: function() { return "Bonjour, je m'appelle " + this.nom; }
};
console.log(personne.saluer());

Explication : Les méthodes sont des fonctions attachées aux objets. this fait référence à l’objet courant.

8. Boucles avancées

const utilisateurs = [
  { nom: "Alice", age: 25 },
  { nom: "Bob", age: 30 }
];
utilisateurs.forEach(u => console.log(u.nom + " a " + u.age + " ans"));

Explication : forEach permet de parcourir un tableau d’objets et d’accéder directement à chaque élément.

9. Math et Date

console.log(Math.random()); // nombre entre 0 et 1
console.log(new Date());    // date actuelle

Explication : L’objet Math fournit des méthodes mathématiques utiles. Date permet de gérer le temps et la date.

10. Getters et Setters

const personne = {
  _age: 25,
  get age() { return this._age; },
  set age(n) { this._age = n; }
};
console.log(personne.age);
personne.age = 30;
console.log(personne.age);

Explication : Les getters et setters permettent de contrôler la lecture et l’écriture d’une propriété, utile pour validation ou calcul.

11. LocalStorage

localStorage.setItem("nom", "Alice");
console.log(localStorage.getItem("nom"));

Explication : LocalStorage conserve les données même après fermeture du navigateur. setItem stocke, getItem récupère.

12. SessionStorage

sessionStorage.setItem("couleur", "bleu");
console.log(sessionStorage.getItem("couleur"));

Explication : SessionStorage stocke des données pour la session en cours. Fermeture du navigateur = perte des données.

13. Filtrage de tableau

const nombres = [1,2,3,4,5];
const pairs = nombres.filter(n => n % 2 === 0);
console.log(pairs);

Explication : filter crée un nouveau tableau en sélectionnant les éléments selon une condition.

14. Map et Reduce

const nombres = [1,2,3,4];
const doubles = nombres.map(n => n*2);
const somme = nombres.reduce((acc, n) => acc + n, 0);
console.log(doubles, somme);

Explication : map transforme un tableau, reduce combine les valeurs en une seule.

15. Classes

class Animal {
  constructor(nom) { this.nom = nom; }
  parler() { return this.nom + " fait un bruit"; }
}
const chat = new Animal("Miaou");
console.log(chat.parler());

Explication : Les classes définissent des objets réutilisables. constructor initialise et les méthodes définissent les comportements.

16. Création d’éléments DOM

const ul = document.createElement("ul");
for(let i=1;i<=5;i++){
  const li = document.createElement("li");
  li.textContent = "Item " + i;
  ul.appendChild(li);
}
document.body.appendChild(ul);

Explication : On crée des éléments dynamiquement et on les ajoute au DOM. Très utile pour générer du contenu avec JS.

17. Événements complexes

const input = document.createElement("input");
const div = document.createElement("div");
document.body.append(input, div);
input.addEventListener("input", () => {
  div.textContent = "Vous avez tapé : " + input.value;
});

Explication : Écouter l’événement input pour chaque frappe et mettre à jour le DOM en temps réel.

18. Mini-calculatrice

function calculer(a,b,op){
  switch(op){
    case "+": return a+b;
    case "-": return a-b;
    case "*": return a*b;
    case "/": return a/b;
  }
}
console.log(calculer(5,3,"*")); // 15

Explication : La fonction prend un opérateur et applique le calcul correspondant via switch.

19. API Fake JSON

fetch("https://jsonplaceholder.typicode.com/todos")
  .then(res => res.json())
  .then(data => console.log(data));

Explication : On utilise fetch pour récupérer des données JSON et les afficher. Idéal pour comprendre l’appel d’API.

20. Mini-projet : Todo List

// Création des éléments
const input = document.createElement("input");
const button = document.createElement("button");
button.textContent = "Ajouter";
const ul = document.createElement("ul");
document.body.append(input, button, ul);

// Ajout d'élément à la liste
button.addEventListener("click", () => {
  if(input.value.trim() !== ""){
    const li = document.createElement("li");
    li.textContent = input.value;
    ul.appendChild(li);
    localStorage.setItem("todos", ul.innerHTML);
    input.value = "";
  }
});

// Récupération depuis localStorage
ul.innerHTML = localStorage.getItem("todos") || "";

Explication : Todo-list entièrement générée par JS. On crée les éléments, ajoute des événements, et on stocke les données avec localStorage. C’est un mini-projet complet pour comprendre DOM, événements et stockage local.
