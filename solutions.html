1. Console et variables

let a = 5;
let b = 10;
console.log("Résultat :", a + b);

Deux variables sont déclarées et additionnées. `console.log()` permet d’afficher le résultat.

3. Fonctions

function carre(n){
    return n * n;
}
console.log(carre(5)); // 25

Fonction simple avec paramètre et `return` pour renvoyer le carré.

4. Manipulation du DOM

document.querySelector("#monParagraphe").textContent = "Nouveau texte";

On cible un élément et on modifie son contenu avec `textContent`.

5. Événements

document.querySelector("#monBouton").addEventListener("click", () => {
  alert("Bouton cliqué !");
});

On écoute le clic du bouton avec `addEventListener` et on déclenche une alerte.
