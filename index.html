Exercice JavaScript - Révision en 20 étapes

Objectif : Réviser les bases de JavaScript et créer un mini-projet dynamique uniquement avec JS et CSS.

Instructions générales :

  • Créer un fichier index.html, style.css et script.js.
  • Ne pas écrire de contenu HTML statique : tout doit être généré via JavaScript.
  • Réaliser les exercices dans l'ordre pour progresser du niveau facile au difficile.
  • Utiliser la console pour tester vos variables, boucles, fonctions et objets.
  • À la fin, relier votre mini-projet au CSS pour un rendu esthétique.

Étapes de révision

  1. Console et alert : Afficher des messages dans la console et avec alert().
  2. Variables : Déclarer let, const, var et manipuler leurs valeurs.
  3. Types de données : String, Number, Boolean, null, undefined.
  4. Tableaux : Création, accès aux éléments, ajout/suppression avec push/pop/shift/unshift.
  5. Objets : Créer un objet, accéder à ses propriétés et méthodes.
  6. Boucles : for, while, for...of.
  7. Conditions : if/else, switch.
  8. Fonctions : Déclaration, expression, fléchées, paramètres et retour de valeur.
  9. DOM - sélection : getElementById, querySelector, querySelectorAll.
  10. DOM - modification : textContent, innerHTML, style.
  11. Événements : addEventListener sur click, input, submit.
  12. Formulaires : Récupérer les valeurs des inputs et valider les données.
  13. Objet Math : Math.random(), Math.floor(), Math.round().
  14. Objet Date : Créer des dates, récupérer jour, mois, année, heure.
  15. Getters et setters : Créer des objets avec méthodes get et set.
  16. LocalStorage et SessionStorage : Stocker, récupérer et supprimer des données.
  17. JSON : Conversion JSON.stringify() et JSON.parse().
  18. Niveau intermédiaire : Créer une petite liste interactive : ajouter, modifier et supprimer des éléments via JS.
  19. Niveau difficile : Générer tout le contenu HTML avec JS pour un mini tableau ou galerie.
  20. Projet final : Créer un mini-projet : liste de films, galerie de produits ou todo-list entièrement générée par JS et stylisée avec CSS.

Objectifs pédagogiques

  • Maîtriser la console, les variables, tableaux, objets et fonctions.
  • Manipuler le DOM et gérer les événements.
  • Comprendre le stockage web (localStorage/sessionStorage).
  • Passer d’un simple script JS à un projet dynamique complet.
  • Préparer le passage à un backend Express et à des appels API.

Astuce : Commencez par des petits exercices pour chaque étape, puis combinez-les progressivement pour créer votre mini-projet final.
